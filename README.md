# Script SYS1

## Download the script
```bash
    git clone https://github.com/Anerti/SYS.git
```

## Requirement:
you need to install neofetch, run the following command
```bash
    sudo apt-get update && sudo apt install neofetch
```
## Run the script
```bash
    chmod +x mon_menu.sh && ./mon_menu.sh
```
